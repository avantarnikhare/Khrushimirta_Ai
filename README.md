**KrushiMitra AI** is an advanced, production-ready intelligent farming assistant designed to empower farmers with data-driven insights and AI-powered guidance. This platform bridges the gap between traditional agriculture and modern technology, built with a focus on high performance, scalability, and an immersive user experience.

---

## 🚀 Deep Project Architecture

KrushiMitra AI is more than just a dashboard; it is an intelligent ecosystem that converts raw data into **Actionable Intelligence**. 

### 🧠 Core Intelligence:
- **AI Expert Advisor:** Powered by the **Groq LPU (Language Processing Unit)**, providing near-instant inference for complex agricultural queries.
- **Contextual Awareness:** The system doesn't just provide generic answers; it analyzes local soil pH, weather patterns, and regional data to offer tailor-made farming solutions.

---

## ✨ Key Features

* **🤖 AI Farming Advisor:** Instant solutions for crop diseases, pest control, and soil management using advanced LLMs.
* **🌤️ Real-time Weather Dashboard:** Hyper-local forecasts and agricultural weather alerts integrated via **OpenWeatherMap API**.
* **📍 Location-Based Insights:** Utilizes **Geoapify** to suggest the best-suited crops based on specific geographical and climatic conditions.
* **⚡ Modern Tech Stack:** Built with **Next.js 14 (App Router)** for optimized Server-Side Rendering (SSR) and lightning-fast page loads.
* **📱 Responsive Design:** Fully optimized for mobile, tablet, and desktop devices using **Tailwind CSS**.

## 🪄 3D Animations & Immersive UI

To make this project stand out on GitHub, I implemented advanced motion design principles:

1.  **3D Perspective Cards:** Using `rotateX` and `rotateY` transforms within **Framer Motion**, cards respond to mouse hover with a realistic 3D tilt effect.
2.  **Staggered Page Transitions:** Elements enter the viewport in a timed sequence, creating a high-end "staggered" reveal effect.
3.  **Glassmorphism:** The UI utilizes translucent backgrounds with high-intensity blurs, providing a sleek, modern, and premium aesthetic.
4.  **Floating Micro-interactions:** Buttons and input fields feature haptic-like animations to improve the tactical feel of the application.

## 💻 Installation & Setup

To run this project locally:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/avantarnikhare/Khrushimirta_Ai.git](https://github.com/avantarnikhare/Khrushimirta_Ai.git)
    ```
2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Setup (`.env.local`):**
    ```env
    GROQ_API_KEY=your_key_here
    OPENWEATHER_API_KEY=your_key_here
    GEOAPIFY_API_KEY=your_key_here
    ```
4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
